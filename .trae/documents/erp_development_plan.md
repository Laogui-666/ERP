# ERP项目开发计划

## 项目现状分析

### 项目结构
- **前端**: Next.js 15.5.14 + React 19.2.4 + Tailwind CSS 3.4.16
- **后端**: Next.js API Routes + Prisma ORM + MySQL
- **实时通信**: Socket.io 4.8.1
- **状态管理**: Zustand 5.0.2
- **数据库**: MySQL (阿里云RDS)
- **文件存储**: 阿里云OSS

### 已实现功能
- 用户认证系统 (登录、注册、重置密码)
- 订单管理系统
- 聊天系统
- 文档管理系统
- 分析仪表盘
- 多角色权限管理 (管理员、客户、门户用户)

### 项目依赖
- `@prisma/client`: ^5.22.0
- `next`: ^15.5.14
- `react`: ^19.2.4
- `socket.io`: ^4.8.1
- `recharts`: ^3.8.0
- `zod`: ^3.24.1
- `zustand`: ^5.0.2

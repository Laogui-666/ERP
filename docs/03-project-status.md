## 5. 风险与待办

### 5.1 已识别风险

| 风险 | 等级 | 影响 | 应对方案 |
|---|:---:|---|---|
| backdrop-blur 性能 | 中 | 低端设备卡顿 | CSS 降级方案 |
| RDS 连接数限制 | 中 | 高并发连接耗尽 | Prisma 连接池配置 |
| Next.js 14.2.18 安全漏洞 | 中 | 多个已知 CVE | 待升级到最新稳定版 |
| Prisma db push 会误删旧表 | 高 | 其他项目数据丢失 | 已发现并改用 SQL 直接建表 |
